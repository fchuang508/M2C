Co-Art Create Promotion · Dao 南塘艺术共创营（第三期）
本约定记录的，是一群朋友为「艺术共创营系列宣传框架」这件共同的事，各自拿出手上不一样的资源——算力、空间、劳务、方法、素材——彼此交换、共同建造。我们以此打破各自为政的原子状态，把个体编织进一个可以延续的社群。在这份约定里，每一位都是平行的共创人，而非彼此的甲方或乙方；出资不等于出钱，积分只是资源的一种计量，不是价款。

◆ 主动议原文（引用）
一、以艺术共创营系列宣传框架为基础，形成自适应窄屏（手机优先）的呈现方案，覆盖第一、二、三期，纳入已办内容及第四期即将举办的内容，以 prototype 方式迭代；工作范围限定为第三期基础背景完成。

二、各期页面基础结构与链接点：一、二期以过往作品集与画廊呈现为主；第三期含画廊与「议事」成果。Level one 页面作为进入基础页；页面档头介绍共创营是什么、缘起、历程、当前状况、伙伴关系，并拉入砚仁系列共创。

三、目标构建「100 位共创人」基础，为第四期宣传承接。

四、推进周期：6 月 22 日开始，至 23 日晚前完成，约 24 小时工作当量，三天周期。

五、授权与资源支持：提案人（FC）提供有限 AI 工具与能力（ChatGPT、Cloud Code、Propensity API、Talk API Key 等）、域名、固定 IP、网页空间；预算限定为共同认同的积分 1200 个南塘豆及 600 个萝卜丁，打给淑惠（支付数额以会后进一步修正为准）。
✓ 决议结果：2:0 通过 跳跳 · 1票 · 赞成 和光同尘 · 1票 · 赞成 反对：0票
为什么把它做成共创约定，而不是报价单：各人投入的资源性质不同（资金/资源、劳务、方法、素材），换回的权益也不同（所有权、使用权、积分回报）。这是一次共同交换，目的在于把原子化的个体织进可延续的社群——这一点比任何单笔交易都更重要。

▌ 共同约定
  1. 共创人地位平等。各自的投入与权益均摊于本约定，对社群公开存证；本约定收入 M2C 公共仓库，即为公开的一部分。
  2. 代码库所有权与后续迭代权归富章（FC）持有；淑惠、跳跳、砚仁享有使用权。各方使用权的后续细分，由相关共创人接下来自行商定。
  3. 成果属于本次共创社群；各共创人的贡献均予署名记录；后续迭代向社群开放协作。
  4. 预算所列积分（1200 南塘豆 + 600 萝卜丁）为暂定值，以会后最终修正为准。
  5. 本次共创为社群之起点而非终点：承接「100 位共创人」目标，为第四期宣传铺垫；未来新的共创人可在同一约定精神下加入。
风险与待补：本约定项下之法律风险，尤其大陆法律体系下的合规性，沿用历次会议之标注，保留为待评估事项；金额、使用权细分等以「待社群后续修正」处理，不在此硬性锁定。详见「备注 · 引用」分页。
